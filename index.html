Links to versions of "square root iteration" in 4 languages

For pseudocode, click to see the question which prompted this site.

The repository for this site is at: https://github.com/bwattle/SquareRoot

A Visual Studio (VS) ZIP file with all 4 languages, ready to load to GitHub is at: Release v1.02

This is an interesting problem as it covers:
  • a while loop which does not count
  • errors with floating point decimals
  • string concatenation with carriage returns and line breaks (v1.02)
  • different rounding functions (v1.03)
  • infinite loop (runtime error) - caused with more than 6 decimal places, VB and Pascal. 16 decimal places for JS and Python (double precision)
  • specific declarations of "single" or "double" can be done specifically in Pascal and VB
    • Pascal: https://wiki.freepascal.org/IEEE_754_formats
    • VB: https://docs.microsoft.com/en-us/dotnet/visual-basic/language-reference/data-types/
  • declarations of "byte", "Int16", "Int32" and "Integer" can be done in VB
  • validation errors

Additional points:
  • The Pascal files are included in Visual Studio so they upload to GitHub, but they are edited with Pascal XE.
  • The GitIgnore file has exclusions for the .exe files and .zip fiels within the Pas folder.
  • Copying VB folders and changing names to produce an exe with the version number is a nasty process and unnecessary.
  • Python files can be run in IDLE, but break points can be done in VS.
  • HTML/JavaScript has been in the inspector of the Chrome browser, with break points being used when necessary.
  • All Git pushing and pulling is done through the "Git Changes" tab of VS.
